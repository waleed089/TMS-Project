
Below are the instructions to build and deploy;

* Building TMS image;
	> open terminal
	> goto to project directory with below command, example "D:/Project/tms
	> cd D:/Project/tms
	> then execute below command
	> docker-compose build
	> after building successfully, execute below command
	> docker-compose up
	
* For Api documentation you can access by below url:
	http://localhost:8080/api-docs
